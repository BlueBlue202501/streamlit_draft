# Streamlit Draft - 字元儀表板

政府AI應用實驗站字元使用量監控儀表板

## 特色功能

- 🎨 深藍色主題配色 (Tailwind CSS)
- 📊 三區字元額度總覽卡片 (第一年度、第二年度、自購)
- 📈 互動式圖表 (使用者排行榜、Bot排行榜、時間趨勢)
- ⚡ 動畫效果與懸浮效果
- 📱 響應式設計
- 🔄 動態數據篩選器
